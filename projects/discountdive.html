Overview

Discount Dive is a cooperative game prototype created as part of a course project simulating a publisher pitch. The experience focuses on physics-driven character interactions using active ragdolls, promoting frantic, emergent co-op gameplay. The project deliverables included a playable prototype, trailer and a pitch document.

Implementation

Prototype built in Unity with custom active ragdoll controllers and physics-driven combat. Emphasis was placed on responsive feel and predictable chaos for cooperative play. Additional contributions include 3D modelling and trailer animation.

Role & Contributions

Programming • Modelling • QA

Worked on character physics, combat systems, QA and several modelling/animation tasks. Coordinated with teammates to integrate scenes for the trailer and prepared the pitch materials presented to the course 'publisher'.

Notes

Course project developed by a team of six. Source code and assets are linked in the repository. The prototype demonstrates systems for cooperative, physics-first gameplay that can be expanded into larger prototypes.
