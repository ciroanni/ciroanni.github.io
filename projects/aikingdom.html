Overview

AI Kingdom is a small, whimsical simulation inspired by SimCity and created during a two-day game jam. Gameplay unfolds on a compact 5×5 grid that evolves turn by turn via a cellular automaton. Instead of fixed rule sets, each tile's update is produced by a language model that describes the tile's next state, enabling emergent and narrative-rich world evolution. Players shape the land by placing tiles and writing prompts that guide the AI-driven updates.

Design & Mechanics

Grid • Turns • AI updates

The game runs on a 5×5 grid. Each turn, tiles are evaluated and updated via prompts sent to a language model; the model's textual outputs determine tile transformations, producing a playful narrative-driven simulation rather than deterministic rules.

Implementation

Unity • C# • LLM

The prototype was built in Unity using C#. A lightweight integration with a language-model backend drives tile updates.

Notes

This project is a prototype from a short game jam and demonstrates an exploratory approach to combining LLMs with game systems. Source code and materials are available on the project's GitHub repository linked above.
