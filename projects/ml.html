3D Object Reconstruction

A simplified RNN-inspired approach to reconstruct 3D objects from multiple images (voxels), implemented in Python and PyTorch.

Overview

This academic exercise explores a reduced-parameter implementation inspired by the 3D-R2N2 architecture to reconstruct voxelized 3D shapes from multiple input images. The goal was to study behaviour and trade-offs when simplifying the original architecture and lowering computational requirements; it does not introduce novel research contributions.

Implementation

Python • PyTorch • RNN

The prototype uses a recurrent encoder to integrate multi-view image features and a decoder that predicts voxel occupancy. Training and evaluation scripts, data preprocessing, and example notebooks are included in the shared project folder.

Scope & Limitations

This work intentionally reduces model complexity and compute to enable experimentation within limited resources. It is a learning exercise and is explicitly inspired by 3D-R2N2; it does not claim methodological novelty.
