Screen Grabbing Utility

Rust utility to capture the display, post-process frames and export results in multiple formats.

Overview

This project provides Rust code to capture the current contents of a display, perform optional post-processing (e.g. resizing, color adjustments) and export the output in one or more file formats. It is designed for performance and portability across desktop platforms.

Implementation

The utility uses native OS capture APIs wrapped in Rust, with image manipulation performed using Rust image crates. Focus is on efficient frame acquisition and deterministic post-processing pipelines suitable for downstream consumers.

Usage

Repository includes build instructions, example commands and sample scripts to save captures to disk or pipe them into processing pipelines.
